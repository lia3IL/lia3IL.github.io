Loading custom styled geoJSON data Points using AJAX: Hurricanes

This example retrieves and parses geoJSON data holding information associated with the storms of the 2004 Hurricane Season. The file is loaded using AJAX. The color of each data point is associated to the legend. The size of each data point is based on the wind speed.

The JSON file is located at: data/hurricanes.geojson

2004 Hurricane Season

The control can be loaded using the following <script> element: 

<script type="text/javascript" charset="UTF-8" src="libs/geojson-parser.js"></script>
